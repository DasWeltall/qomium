# Qomium - Advanced Meta Search Engine

**Qomium** is an advanced meta search engine project that leverages [SearXNG](https://github.com/searxng/searxng) to provide fast and flexible search capabilities.  
With Qomium, you can search for images, videos, news, and much more—all in one place.  
Additionally, Qomium can generate AI-powered summaries of your search results using the Groq API.

## Features

- **Multi-Category Search:** Find images, videos, news, and more, powered by SearxNG.
- **Privacy-Focused:** Qomium does not store user data or search history.
- **AI Summaries:** Get concise, AI-generated summaries of your search results.
- **Customizable:** Easily configure your search experience and summary features.

## AI Summaries with Groq

To enable AI summaries, you need to add your Groq API key in the settings:

1. Go to [console.groq](https://console.groq.com/) and create an API key.
2. Open Qomium’s settings and enter your Groq API key.
3. Now, every search result can be summarized using AI!

## How It Works

- Enter your search query.
- Select the category (web, images, videos, news, etc.).
- Qomium fetches results from multiple sources using SearXNG.
- (Optional) Enable AI summary for a quick overview of the results.

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** SearxNG meta search engine
- **AI Integration:** Groq API for summaries

## Getting Started

Qomium can be used on a server with PHP installed. 
To enable AI summaries, make sure to add your Groq API key in the settings.
